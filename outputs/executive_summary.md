# Executive Summary

This Streamlit project gives Nassau Candy Distributor a decision-support system for factory reallocation and shipping optimization. It learns historical lead-time behavior, simulates alternate factory assignments, and ranks recommendations based on efficiency gain, profit stability, confidence, and reassignment risk.

## Recommended Use

- Use the Factory Simulator to test one product under a selected destination region and ship mode.
- Use the Recommendations dashboard to prioritize high-scoring reallocation opportunities.
- Use the Route Clusters view to identify consistently slow or congested route patterns.
- Treat high-risk recommendations as candidates for manual validation before operational rollout.

## Business Value

The system moves the factory assignment process from static rules to evidence-based scenario planning. Leadership can compare current and alternate configurations before execution, helping reduce avoidable lead time while protecting gross profit.
